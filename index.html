How to use layout_trainer...
Hello and welcome to layout_trainer, a keyboard layout training application for those who wish to increase their typing speed and accuracy on alternate keyboard layouts. (or Korean 한글)

To begin using this tool, please type the displayed text into the input field under the text block.

As you begin typing the timer above the input field will start counting.

If you wish to restart or get a new set of text displayed, click the refresh button below the input field.

The virtual keyboard displays the current layout selected, and responds to your key presses.

The default starting layout is Colemak.
If you wish to change to a different layout, click the > right arrow located under the keyboard and next to the current layout name.

The currently offered layouts are: 1. Colemak, 2. Korean (한글), 3. Dvorak

When you’ve completed a block, your words per minute (or characters per minute for Korean) will be calculated for you.

You must get 100% accuracy to complete a block.

Thank you for reading!

기본 키보드 레이아웃은 콜맥입니다.
